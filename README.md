# c-envy-free-matching


##  Requirements or Dependencies

```bash
pip install -r requirements.txt
```

or

```bash
rand = "0.6"
pathfinding = "4.4.0"
```
For $\mathrm{pathfinding}$, see https://docs.rs/pathfinding/latest/pathfinding/.


## Usage

```bash
python main.py  
```

or

```bash
cd envy
cargo run  
```
